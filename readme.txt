== Description ==
### Description

Importiere deine Termine von [Evangelische Termine](https://www.evangelische-termine.de/) in das kostenlose Plugin [The Events Calendar](https://wordpress.org/plugins/the-events-calendar/)

### Features:
* Importiere Termine von verschiedenen Veranstaltern
* Importiere Termine aus verschiedenen Regionen

== Installation ==
MINIMALE SYSTEMANFORDERUNGEN
* WordPress 5.6 oder höher
* PHP Version 7.2 oder höher
* MySQL Version 5.6 oder höher
* WordPress Memory Limit von 64 MB oder mehr (128 MB oder höher wird empfohlen)

INSTALLATION
1. Installieren Sie das Plugin mit dem in WordPress integrierten Plugin-Installationsprogramm oder extrahieren Sie die ZIP-Datei und legen Sie den Inhalt im Verzeichnis wp-content/plugins/ Ihrer WordPress-Installation ab.
2. Aktiviere das Plugin über den Menüpunkt ‚Plugins‘ in WordPress.
3. Gehe zu Ev. Termine
4. Füllen Sie die Import Einstellung aus und klicken sie auf \"Speichern\". Der Import startet automatisch.
5. Wenn Sie eine Premium Lizenz haben fügen sie den Lizenzschlüssel in den Lizenzeinstellungen ein und klicken Sie auf \"Speichern\"
